TCP Chat — Multi-threaded Server/Client
Low-level multi-threaded TCP chat system in C with dual IPv4/IPv6 stack support. One pthread per connected client, mutex-guarded shared state, and ANSI-colored broadcast messaging — demonstrating real-time networking fundamentals that underpin robot communication.
  • Dual-stack via getaddrinfo(AF_UNSPEC) — handles both IPv4 and IPv6 transparently
  • One pthread per client with pthread_detach; swap-with-last O(1) removal from client table
  • pthread_mutex_t-guarded client registry — race-free broadcast to all peers
  • Per-client ANSI color assignment; formatted [Name]: msg echoed to all others
  • Separate send/receive threads on the client; clean disconnect on EOF (Ctrl-D)
  • SO_REUSEADDR for fast restart; supports up to 100 concurrent clients
